js复制网页内容时自动加文章地址版权

2018-09-23
JavaScript

这是个用金钱堆砌的互联网百花齐放的信息时代，也是个不注重版权的盗版时代。而基本上每个前端er都或轻或重的搞了个自己的博客，如何在别人复制你网站文章内容时，自动在内容的尾端添加一段版权文本加当前页面的链接用以保护我们的权益被摆上日程。

功能代码，不多解释：

<script type='text/javascript'>
function addLink() {
    var body_element = document.getElementsByTagName('body')[0];
    var selection;
    selection = window.getSelection();
    var pagelink = "<br /><br /><br />著作权归bigkaiwu个人博客所有<br />商业转载请联系作者获得授权，非商业转载请注明出处.<br />本文来源：<a href='"+document.location.href+"'>"+document.location.href+"</a>";
    var copy_text = selection + pagelink;
    var new_div = document.createElement('div');
    new_div.style.left='-99999px';
    new_div.style.position='absolute';
    body_element.appendChild(new_div );
    new_div.innerHTML = copy_text ;
    selection.selectAllChildren(new_div );
    window.setTimeout(function() {
        body_element.removeChild(new_div );
    },0);
}
    document.oncopy = addLink;
</script>

又或者你心狠一点，直接把复制禁用掉：

<script> 
function noCopy() 
{ 
    event.returnValue = false; 
} 
</script> 
<body oncopy = "noCopy()">

或者，心地善良的你选择给个禁止复制的小提示：

<body oncopy="alert('对不起，禁止复制！');return false;">

然后，你突然又想复制失效的同时，让右键也一起禁用掉，可以使用下面的代码：

<script language="Javascript"> 
document.oncontextmenu=new Function("event.returnValue=false"); 
document.onselectstart=new Function("event.returnValue=false"); 
</script>
